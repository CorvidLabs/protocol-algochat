AlgoChat Protocol

End-to-end encrypted messaging on Algorand

Features

End-to-End Encryption

X25519 + ChaCha20-Poly1305

Forward Secrecy

Per-message ephemeral keys

Immutable Storage

Messages on blockchain

Decentralized

No central server

Wire Format (v1)

version: 1 protocol: 1 sender_pubkey: 32 ephemeral_pubkey: 32 nonce: 12 encrypted_sender_key: 48 ciphertext: variable

Header: 126 bytes | Max message: 882 bytes

Cryptographic Primitives

FunctionAlgorithm
Key AgreementX25519 ECDH
EncryptionChaCha20-Poly1305
Key DerivationHKDF-SHA256

Transport

Messages are transmitted as the note field of Algorand payment transactions:

  • Minimum payment: 0 ALGO
  • Sender pays tx fee (~0.001 ALGO)
  • Note field limit: 1024 bytes
